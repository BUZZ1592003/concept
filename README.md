# Aurelia Spaces

Premium luxury interior design landing page built as a fast static site with responsive sections, cinematic hero transitions, portfolio filtering, testimonial slider, modal project details, and lead-generation CTAs.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
